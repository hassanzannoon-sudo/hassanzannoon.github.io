SIEM & Detection Engineering (Wazuh)

By Hassan Zannoon — SIEM deployment, detection engineering & incident reporting

Executive summary: Built a functional Wazuh SIEM environment, onboarded Windows & Linux endpoints, and developed custom detection rules and dashboards to identify brute-force, privilege escalation, and suspicious PowerShell activity. Focus was on alert fidelity, triage efficiency and MITRE ATT&CK mapping.

Technical implementation (short)

ComponentRole / Notes
Wazuh ManagerCentral analysis, rule processing, alerting and dashboards (Ubuntu server)
Wazuh AgentsInstalled on Windows & Linux endpoints for telemetry collection
ForwardingWinlogbeat/Filebeat for additional log shipping (optional)

Key detections & MITRE mapping

  • Brute-force login attempts — T1110
  • New local admin creation — T1078 / T1543
  • Suspicious PowerShell execution — T1059.001
  • AV tampering — T1562
